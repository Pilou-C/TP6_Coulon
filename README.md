# TP6 - Mastermind (FR)

## Instructions
- Ce TP est à faire individuellement.
- Le sujet est disponible dans `files/tp6.pdf`.
- Les fichiers à compléter/modifier sont `mastermind.c`, `dispay.c` et `main.c`.

## Fichiers à ne pas modifier
- `utest.h` : librairie de tests unitaires.
- `test_mastermind.c` : définitions des tests unitaires pour la partie 1.
- `test_display.c` : définitions des tests unitaires pour la partie 2.
- `run_tests.sh` : script bash pour lancer tous les tests unitaires.
- `CMakeLists.txt` : fichier de configuration pour la compilation des fichiers sources et pour la production des exécutables.

## Tests et validation
- Il est recommander de tester chaque fonction après avoir écrit sa définition.
- Les exécutables `test_mastermind` et `test_display` exécutent les tests unitaires associées respectivement aux parties 1 et 2 du TP.
- L'exécutable `All Ctest` exécute tous les tests unitaires (**à faire une fois que toutes les fonctions sont définies**)
- Le script `run_tests.sh` exécute également tous les tests unitaires, il peut être utilisé en alternative à `All Ctest`.

## Soumission du TP

**Une fois tous les tests passés en local, poussez votre branche principale sur le dépôt GitHub associé au TP**.

# TP6 - Mastermind (EN)

## Instructions
- This lab is to be done individually.
- The subject is available in `files/tp6-en.pdf`.
- The files to be completed/modified are `mastermind.c`, `dispay.c` and `main.c`.
  
## Files not to be modified
- `utest.h` : unit testing library.
- `test_mastermind.c` : definitions of unit tests for part 1.
- `test_display.c` : definitions of unit tests for part 2.
- `run_tests.sh` : bash script to run all unit tests.

## Tests and validation
